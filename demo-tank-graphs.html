Tank Level Graph Options

Option A — Chart.js (library)

Pros: Polished look, easy tooltips, built-in animation, gradient fill trivial.
Cons: Requires CDN load (~60 KB), slight dependency risk if offline.

Option B — SVG adaptive Y-axis

Pros: Zero deps, data fills full vertical space so changes are very readable.
Cons: Y-axis is zoomed in, so it may look misleading if someone expects 0–100%.

Option C — SVG full range (0–100%) with colour zones

Pros: Instantly shows tank is "mostly full", coloured zones give context at a glance.
Cons: Small fluctuations look flat when zoomed to full 0–100% scale.

Option D — Minimal sparkline (FarmBot style)

Pros: Clean, professional, works in light-themed UI, very readable data line.
Cons: Light theme contrasts with the rest of the dark dashboard.
