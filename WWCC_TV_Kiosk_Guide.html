What Is the TV Display?

Purpose, who it is for, and how to access it

The TV/Kiosk display is a fullscreen, auto-rotating dashboard designed to be shown on monitors, TVs, and tablets throughout the club — entrance foyers, the pro shop, member lounges, and marshal stations. It provides at-a-glance course and weather conditions without requiring anyone to navigate a website.

It uses the same live data as the Golfer and Greenkeeper pages — the same Davis WeatherLink station, the same Open-Meteo forecasts — but reformatted for large-screen passive viewing. Text is larger, layouts are optimised for 16:9 monitors, and the display rotates automatically so staff don't need to touch it.

📺
How to Access: Add ?tv=1 to the end of the weather dashboard URL.
Example: https://bidgee182.github.io/wwcc-weather-page/?tv=1
This hides the normal navigation tabs and activates the fullscreen TV layout.

Slide 1 — Current Weather

Playing conditions banner and 8 live condition tiles

Playing Conditions Banner

The large coloured banner at the top of Slide 1 gives the overall playing conditions rating. It is calculated by a penalty-point system — the dashboard starts at zero and adds points for each factor that worsens conditions (temperature extremes, wind, rain, UV, course wetness, daylight). The total score determines the rating:

The 8 Condition Tiles

Eight tiles display in a 4×2 grid, each colour-coded from green (good) through yellow, orange and red (bad), with blue for wet conditions and purple for extreme UV. The Dew Point tile is TV-only — it replaces the Course Roll tile which moves to Slide 2.

Slide 2 — Golf Conditions

Insight cards, green speed, ball flight, and wind hole analysis

Course Roll Card

The first card shows how firm or soft the course is underfoot. Calculated from the 7-day water balance (cumulative ET minus cumulative rainfall). This is a TV-exclusive card — on the golfer page the same information is shown as a tile.

7-Day BalanceStatusWhat Golfers Notice
< −10 mm surplusWaterloggedCarts on paths; possible casual water
−10 to 0 mmSoftGood grip; approaches hold well
0 to +15 mmNormalPlaying true; ideal ball roll
+15 to +30 mmFirmBall running on; allow extra run
> +30 mmVery FirmHard and fast; significant run through greens

Best Time to Play Today

Scans every hour between sunrise and one hour before sunset. Each hour is scored from 100 points with penalties deducted for temperature extremes, rain probability, UV, wind strength, low light, and wet course conditions. The best consecutive 3-hour block is found and displayed with a quality rating (Ideal / Good / Acceptable / Best Available).

Green Speed Card

Uses the same 7-day water balance to estimate how fast the greens are running. Wetter = slower and more receptive; drier = faster and harder to hold. Ranges from "Slower than usual (Soft & Receptive)" through "Normal (Running True)" to "Faster than usual (Fast & Bouncy)".

Ball at 150m Card

Calculates how far the ball will actually carry in current atmospheric conditions compared to a standard 150m shot:

Playing distance = 150m + (Temperature − 20°C) × 0.20m warmer = farther + (1015 − Pressure hPa) × 0.03m lower pressure = farther + (RH% − 50) × 0.005m minor humidity effect

The difference from 150m determines whether to club up, club down, or play normally. Wind is shown separately as it varies by hole.

Wind Holes Analysis — TV Exclusive

This section is unique to the TV display. Using the GPS coordinates of all 18 holes stored in the dashboard, it calculates the angle of each hole relative to the current wind direction and identifies the three most significant wind-affected holes:

Feature BoxWhat It Shows
Hardest Into WindThe hole most directly into the headwind — includes hole number, distance impact in metres, recommended club change, green behaviour notes
Best TailwindThe hole with the best tailwind assist — how many metres farther the ball will carry, club adjustment
Worst CrosswindThe hole with the worst lateral drift — how many metres sideways, which direction to aim, wind source direction
📍
The wind hole calculations use the GPS coordinates of every tee, fairway midpoint, and green for all 18 holes, combined with the current live wind bearing and speed from the Davis station. The hole analysis updates automatically as wind direction changes.

Next 5 Hours — Hourly Forecast Strip

Shows the next 5 hours from now using Open-Meteo hourly forecast data. Each hour card shows: weather icon, temperature (°C), rain probability (%), UV index, and wind speed (km/h). The first card is labelled "Now" and uses live station data.

4-Day Forecast

Four day cards showing: day name, weather icon, plain-English description (Clear, Partly Cloudy, Rain, Thunderstorm etc.), high and low temperatures, total rain (mm), rain probability (%), and peak UV index. Sourced from Open-Meteo daily forecast, updated every 30 minutes.

Alert Banners

Four automatic warnings shown above all slides when triggered

Alert banners appear above the slide carousel and are visible on all three slides simultaneously. Multiple alerts can be active at the same time and stack vertically. They disappear automatically when the condition clears (checked every 15 seconds).

⚡ Thunderstorm Alert
Trigger: Thunderstorm (WMO code 95/96/99) forecast in next 4 hours
Style: Pulsing amber glow
Action: Suspend play — lightning risk in area
❄️ Frost Alert
Trigger: Overnight low < 4°C (8 PM – 9 AM window)
Style: Pulsing blue glow
Action: Delay early tee times; hold carts on paths
🌫️ Fog Alert
Trigger: Fog forecast tomorrow morning (WMO code 45 or 48)
Style: Static grey bar
Action: Check visibility before early tee times
🛺 Carts on Paths
Trigger: 7-day surplus > 10mm OR 24h rain ≥ 15mm
Style: Static amber bar
Action: Enforce paths-only policy for all carts
⚡
The Thunderstorm Alert is the highest priority. It pulses to draw attention even from a distance. When this banner is active, pro shop staff should be advised immediately and play suspension procedures should be followed per the club's lightning policy.

🔄
Auto-update: The page checks for new versions of itself every 5 minutes. If the dashboard HTML has been updated (e.g. new features added), the TV display will automatically reload to pick up the changes — no staff action needed.
📶
Offline behaviour: If the internet drops, the display continues showing the last fetched data. The "last updated" timestamp in the footer turns red after 10 minutes of stale station data. Forecast data remains visible from the last successful fetch.

Setup & IT Instructions

How to install and configure the TV display

Accessing TV Mode

🔗
Standard dashboard:
weather.html

TV/Kiosk mode — add ?tv=1 to the URL:
weather.html?tv=1

Step-by-Step Setup

1
Connect the display device
Connect a PC, mini-PC, Chromecast, or tablet to the TV/monitor via HDMI. Alternatively use the screen's built-in browser if it supports modern HTML5.
2
Open the dashboard in a modern browser
Use Google Chrome, Microsoft Edge, or Mozilla Firefox. Open the weather.html file and add ?tv=1 to the end of the URL in the address bar.
3
Go fullscreen
Press F11 to enter fullscreen mode and hide the browser address bar. The dashboard is designed to fill the entire screen with no scrolling.
4
Ensure internet connectivity
The display requires internet access to fetch live station data and forecasts. Connect the device to the club WiFi or a wired ethernet connection. Check the "last updated" indicator stays green.
5
Set the display to stay on
Disable screen sleep/screensaver on the device. On Windows: Settings → Power → Sleep → Never. On Mac: System Preferences → Energy Saver → turn off display sleep. On tablet: set auto-lock to Never.
6
Optional: Set browser to open automatically
On Windows: add a shortcut to the browser with the URL to the Windows startup folder (shell:startup). This ensures the display comes back up automatically after a power outage.
